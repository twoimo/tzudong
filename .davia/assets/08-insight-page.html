📊 인사이트 페이지

관리자 전용 "쯔동여지도 인사이트" 페이지 구현 문서입니다.

📅 최종 수정: 2025-12-12 | 진행률: 약 60%

🎯 개요

세 가지 핵심 분석 기능을 제공합니다:

  • 유튜브 히트맵 분석 - 영상별 시청자 참여도 시각화
  • 타 유튜버 맛집 지도 - 네이버 지도 기반 마커 표시
  • 리뷰 워드 클라우드 - 음식 키워드 시각화 (밥그릇 모양)

🔧 기술 스택

📈 구현 상태

🎨 섹션별 상세

1. 유튜브 히트맵 (HeatmapSection)

  • 영상 목록 - 6개 모의 데이터 영상
  • 히트맵 차트 - recharts AreaChart + 그라데이션
  • AI 분석 - Peak/Low 구간 원인 분석, 핵심 키워드 태그
  • 통계 - 조회수, 게시일, 주간 변화율(%)

2. 맛집 지도 (MapSection)

  • 좌측 패널 - 검색 + 필터 (전체/사용자 제보/타 유튜버)
  • 중앙 지도 - NaverMapView 컴포넌트
  • 마커 스타일 - 사용자 제보(파랑), 타 유튜버(빨강)
  • 우측 패널 - 선택된 맛집 상세 + 유튜브 링크

3. 워드 클라우드 (WordCloudSection)

  • 시각화 - 밥그릇(🍚) 모양 SVG 내부에 키워드 배치
  • 데이터 - 수십 종 음식 키워드 (트렌드, 카테고리 포함)
  • 인터랙션 - 키워드 클릭 → 관련 영상/리뷰 표시

⚡ 성능 최적화

기법 적용 위치 효과
next/dynamic 각 섹션 컴포넌트 초기 번들 크기 감소

useMemo 필터링된 데이터 검색/필터 연산 캐싱
staleTime React Query YouTube API 5분 캐싱
