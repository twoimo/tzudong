Quick Start - 5분 안에 시작하기

쯔동여지도를 5분 안에 로컬에서 실행하는 가장 빠른 방법입니다.

사전 요구사항

아래 도구들이 설치되어 있어야 합니다:

  • Node.js 18+ - nodejs.org
  • pnpm - npm install -g pnpm
  • Git - git-scm.com

Step 1: 프로젝트 다운로드 (1분)

# 저장소 클론
git clone https://github.com/twoimo/tzudong.git

# 프로젝트 폴더로 이동
cd tzudong

Step 2: 의존성 설치 (2분)

# 프론트엔드 폴더로 이동
cd apps/web

# 패키지 설치
pnpm install

Step 3: 환경변수 설정 (1분)

# 환경변수 파일 복사
cp .env.example .env.local

.env.local 파일을 열어서 아래 7개 필수 환경변수를 모두 채워주세요:

# Supabase
NEXT_PUBLIC_SUPABASE_URL=https://xxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1...

# Google Maps
NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=your_google_key

# Naver API
NEXT_PUBLIC_NAVER_CLIENT_ID=your_naver_id
NEXT_PUBLIC_NAVER_CLIENT_SECRET=your_naver_secret

# YouTube API
NEXT_PUBLIC_YOUTUBE_API_KEY=your_youtube_key

# OpenAI API
NEXT_OPENAI_API_KEY_BYEON=your_openai_key

Step 4: 개발 서버 실행 (1분)

# 개발 서버 시작
pnpm dev

브라우저에서 http://localhost:3000 접속!

완료!

이제 로컬에서 쯔동여지도가 실행됩니다.

API 키 발급 가이드

Supabase (필수)

  1. supabase.com 접속
  2. 새 프로젝트 생성 (Region: Northeast Asia Seoul 권장)
  3. Settings → API에서 URL과 anon key 복사

Google Maps (필수)

  1. console.cloud.google.com 접속
  2. 새 프로젝트 생성
  3. Maps JavaScript API, Geocoding API 활성화
  4. API 키 생성 및 복사
  5. ⚠️ 빌링 설정 필수 (무료 크레딧 $200/월 제공)

네이버 지도 Geocoding

외부 프록시 서버 사용 시 API 키 불필요:

http://www.moamodu.com/develop/naver_map_new_proxy.php

직접 사용 시:

  1. console.ncloud.com 접속
  2. Maps → Application 등록 → Geocoding 선택
  3. Client ID, Client Secret 복사

YouTube API

  1. Google Cloud Console → YouTube Data API v3 활성화
  2. API 키 생성

더 자세한 환경변수 설정은 환경변수 가이드를 참조하세요.

데이터베이스 마이그레이션

# Supabase CLI 로그인
supabase login

# 프로젝트 연결
supabase link --project-ref your-project-ref

# 마이그레이션 실행
supabase db push

# 마이그레이션 확인
supabase migration list

관리자 계정 설정

회원가입 후 Supabase SQL Editor에서 실행:

-- 사용자 ID 확인
SELECT id, email FROM auth.users WHERE email = 'your-email@example.com';

-- 관리자 권한 부여
INSERT INTO user_roles (user_id, role)
VALUES ('사용자-id', 'admin');

문제가 발생하면? → 트러블슈팅 가이드
