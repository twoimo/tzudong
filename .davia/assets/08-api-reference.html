API 레퍼런스

쯔동여지도에서 사용하는 Supabase API 엔드포인트와 데이터베이스 함수를 정리한 문서입니다.

Supabase 클라이언트 설정

import { createClient } from '@supabase/supabase-js'

const supabase = createClient(
  process.env.NEXT_PUBLIC_SUPABASE_URL,
  process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY
)

맛집 (restaurants)

승인된 맛집 목록 조회

// GET /rest/v1/restaurants?status=eq.approved
const { data, error } = await supabase
  .from('restaurants')
  .select('*')
  .eq('status', 'approved')
  .order('created_at', { ascending: false })
  .limit(50)

  .gte('lat', minLat)

리뷰 (reviews)

리뷰 목록 조회

// 특정 맛집의 리뷰
const { data, error } = await supabase
  .from('reviews')
  .select(`
    *,
    profiles (nickname, avatar_url),
    review_likes (user_id)
  `)
  .eq('restaurant_id', restaurantId)
  .order('created_at', { ascending: false })

리뷰 작성

// POST /rest/v1/reviews
const { data, error } = await supabase
  .from('reviews')
  .insert({
    restaurant_id: 'uuid-xxx',
    user_id: 'uuid-xxx',  // auth.uid()
    title: '정말 맛있어요!',
    content: '삼겹살이 정말 맛있었습니다...',
    rating: 5,
    visited_at: '2025-01-01',
    verification_photo: 'https://storage.xxx/photo.jpg',
    food_photos: ['url1', 'url2'],
    categories: ['삼겹살', '고기']
  })
  .select()

리뷰 수정

// PATCH /rest/v1/reviews?id=eq.{id}
const { data, error } = await supabase
  .from('reviews')
  .update({
    title: '수정된 제목',
    content: '수정된 내용',
    updated_at: new Date().toISOString()
  })
  .eq('id', reviewId)
  .eq('user_id', userId)  // 본인만 수정 가능

리뷰 삭제

// DELETE /rest/v1/reviews?id=eq.{id}
const { error } = await supabase
  .from('reviews')
  .delete()
  .eq('id', reviewId)
  .eq('user_id', userId)  // 본인만 삭제 가능

리뷰 좋아요

// 좋아요 추가
const { error } = await supabase
  .from('review_likes')
  .insert({
    review_id: reviewId,
    user_id: userId
  })

// 좋아요 취소
const { error } = await supabase
  .from('review_likes')
  .delete()
  .eq('review_id', reviewId)
  .eq('user_id', userId)

인증 (Auth)

Google 로그인

const { data, error } = await supabase.auth.signInWithOAuth({
  provider: 'google',
  options: {
    redirectTo: `${window.location.origin}/auth/callback`
  }
})

로그아웃

const { error } = await supabase.auth.signOut()

현재 사용자 조회

const { data: { user } } = await supabase.auth.getUser()

// 또는 세션에서
const { data: { session } } = await supabase.auth.getSession()
const user = session?.user

세션 변경 감지

supabase.auth.onAuthStateChange((event, session) => {
  if (event === 'SIGNED_IN') {
    console.log('로그인됨', session.user)
  } else if (event === 'SIGNED_OUT') {
    console.log('로그아웃됨')
  }
})

맛집 승인

TypeScript 타입

// types/database.ts
export interface Restaurant {
  id: string
  name: string
  lat: number
  lng: number
  category: string[]
  road_address: string | null
  jibun_address: string | null
  youtube_link: string | null
  youtube_meta: {
    title: string
    thumbnail: string
    view_count: number
  } | null
  evaluation_results: {
    taste_score: number
    quantity_score: number
    actual_visit_confirmed: boolean
  } | null
  tzuyang_review: string | null
  status: 'pending' | 'approved' | 'rejected'
  created_at: string
  updated_at: string
}

export interface Review {
  id: string
  restaurant_id: string
  user_id: string
  title: string
  content: string
  rating: number
  visited_at: string
  verification_photo: string
  food_photos: string[]
  categories: string[]
  is_verified: boolean
  created_at: string
  updated_at: string
}

export interface Profile {
  user_id: string
  nickname: string
  email: string
  avatar_url: string | null
  bio: string | null
  created_at: string
  updated_at: string
}
