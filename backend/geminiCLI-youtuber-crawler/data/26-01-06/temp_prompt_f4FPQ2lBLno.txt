# 작업 순서

## 1단계: 지도 URL 직접 접근 (web_fetch 활용 - 최우선)

### 지도 URL이 있는 경우
**web_fetch를 사용하여 지도 URL에서 직접 정보를 추출하세요:**

```
web_fetch(prompt="이 URL에서 음식점 정보(상호명, 주소, 전화번호, 카테고리)를 추출해줘: [지도URL]")
```

#### 지도 URL 유형별 처리
| 지도 유형 | URL 패턴 | web_fetch 활용 |
|----------|---------|---------------|
| 네이버 | naver.me, map.naver.com, place.naver.com | ⭐ 매우 효과적 |
| 카카오 | kko.to, map.kakao.com | ⭐ 매우 효과적 |
| 구글 | maps.google.com, goo.gl/maps | 좌표 직접 추출 가능 |

**web_fetch로 추출 가능한 정보:**
- 정확한 상호명 (검색 없이 확인)
- 도로명 주소
- 전화번호
- 카테고리 (고기, 한식 등)
- 영업시간

## 2단계: 자막/제목/설명 분석

### 2-1. 영상 설명(description) 분석
- 지도 URL 외에도 텍스트로 적힌 상호명, 주소, 연락처 찾기
- 예: "📍 부일숯불 강남점", "위치: 서울 강남구 역삼동"

### 2-2. 자막(transcript) 심층 분석
자막에서 다음 패턴을 찾으세요:
- **가게 이름 언급**: "[01:30] 오늘은 부일숯불에 왔어요"
- **지역/위치 언급**: "[02:00] 여기 강남역 근처인데요"
- **간판 확인 장면**: "[03:00] (간판) 산북동달구지"

### 2-3. 영상 제목 분석
- 제목에 음식점 이름이나 지역 포함 가능
- 예: "부일숯불 삼겹살 솔직리뷰"

## 3단계: 웹 검색으로 보완 (google_web_search)

**web_fetch로 충분한 정보를 얻지 못한 경우에만** 웹 검색 수행:

### 검색 키워드 예시
- "정육왕 [음식점명]" - 유튜버가 방문한 곳인지 확인
- "[음식점명] [지역명] 주소" - 정확한 주소 확인
- "[음식점명] 전화번호" - 연락처 확인

## 4단계: 유튜버 리뷰 요약
- 맛 평가, 분위기, 가격, 추천 메뉴 등
- 자막 타임스탬프 포함 권장

# 새로운 필드 추출 지침

## 영업시간/휴무일 (business_hours, closed_days)
1. **web_fetch**로 지도 URL에서 영업정보 확인
2. **영상 설명**에서 "영업시간:", "휴무:" 패턴 탐색
3. 없으면 `null`

## 폐업 여부 (is_closed)
1. **google_web_search**로 "[음식점명] 폐업" 검색
2. 최근 리뷰나 뉴스에서 폐업 정보 확인
3. 기본값: `false`

## 주차 정보 (parking)
1. **영상 설명/자막**에서 "주차", "발렛", "parking" 키워드 탐색
2. **web_fetch** 결과에서 주차 정보 확인

## 대표 메뉴 (signature_menu)
1. **영상 제목/자막**에서 언급된 메뉴 추출
2. 유튜버가 추천하거나 주문한 메뉴
3. 최대 3개까지

## 가격대 (price_range)
1. 영상에서 가격 언급 확인
2. 메뉴 가격 기준으로 판단

# 전화번호 추출 강화 지침
1. **영상 설명(description)에서 전화번호 패턴 탐색**:
   - 02-XXXX-XXXX, 031-XXX-XXXX, 010-XXXX-XXXX 등
   - "연락처:", "전화:", "☎", "📞" 다음에 오는 숫자
2. **web_fetch 결과에서 phone 필드 확인**
3. **google_web_search로 "[음식점명] 전화번호" 검색**
4. **찾을 수 없으면**: `"phone": null`, `"phone_source": "not_found"`로 설정하고 `reasoning_basis`에 "[전화번호] 영상/검색에서 찾을 수 없음" 기록

# 중요 주의사항
1. **지도 URL이 있으면 web_fetch를 먼저 사용** → 직접 정보 추출이 가장 정확함
2. web_fetch 실패 시 google_web_search로 보완
3. **지도 URL이 없어도** 자막/제목/설명에서 음식점 정보 적극 탐색
4. 음식점이 없는 영상이면 `"restaurants": []`, `"is_restaurant_video": false`
5. 여러 음식점이 나오면 **모두** 추출 (각 URL마다 web_fetch 수행)
6. 확실하지 않은 정보는 confidence를 "low"로 설정
7. 추측하지 말고, 정보가 없으면 null로 설정
8. 모든 응답은 한국어로 작성
