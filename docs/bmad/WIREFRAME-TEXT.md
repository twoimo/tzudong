# Wireframe (Text)

## 1) 일반 사용자 대시보드 `/insights`

### 1.1 Desktop

```text
┌──────────────────────────────────────────────────────────────────────┐
│ Header: 쯔양 데이터 대시보드 | 기간 필터 | 카테고리 필터             │
├──────────────────────────────────────────────────────────────────────┤
│ KPI Row: [영상수] [식당수] [카테고리 Top1] [최근 업데이트]           │
├───────────────────────────────┬──────────────────────────────────────┤
│ Left (Map, 65%)               │ Right (Trend, 35%)                  │
│ - 지도 + 클러스터             │ - 카테고리 분포 바/도넛             │
│ - 마커 클릭 시 카드 오픈      │ - 업로드/수집 추이 라인             │
│ - 카테고리/지역 필터          │ - notSelection 비율(읽기 전용)      │
├───────────────────────────────┴──────────────────────────────────────┤
│ Video Drill-down Table                                              │
│ [썸네일] [제목] [식당수] [평가상태] [상세보기]                        │
└──────────────────────────────────────────────────────────────────────┘
```

### 1.2 Mobile

```text
[Header]
[KPI horizontal scroll]
[Map]
[Category trend]
[Video list]
```

모바일에서는 지도와 리스트를 세로 스택으로 구성하고, 필터는 바텀시트로 노출한다.

## 2) 관리자 대시보드 `/admin/insight`

### 2.1 Desktop

```text
┌──────────────────────────────────────────────────────────────────────┐
│ Header: 운영 인사이트 | 마지막 배치 시간 | 배치 실행상태              │
├──────────────────────────────────────────────────────────────────────┤
│ Funnel: crawling -> selection -> rule -> laaj                       │
│ [1023]      [841/280]    [758]    [756]                              │
├───────────────────────────────┬──────────────────────────────────────┤
│ Failure Board (left)          │ Quality Board (right)               │
│ - notSelection reason top N   │ - review_faithfulness 분포          │
│ - rule falseMessage top N     │ - location_match true/false         │
│ - laaj 누락 video_id list      │ - 카테고리 품질 분포                │
├───────────────────────────────┴──────────────────────────────────────┤
│ Issue Queue Table                                                     │
│ [video_id] [issue_type] [reason] [severity] [검수로 이동]             │
└──────────────────────────────────────────────────────────────────────┘
```

### 2.2 Mobile

```text
[Funnel cards]
[Failure Board]
[Quality Board]
[Issue Queue list]
```

모바일에서는 퍼널을 카드 슬라이드로 보여주고, 이슈 테이블은 중요 컬럼만 남긴 리스트 카드형으로 축약한다.

## 3) 상세 상호작용

- 지도 마커 클릭 -> 우측 상세 카드(또는 모바일 바텀시트)
- 실패 이슈 클릭 -> `/admin/evaluations?video_id=...&issue=...`
- 영상 row 클릭 -> `/insights?video_id=...` 내 modal drill-down

## 4) 디자인 가이드 (이번 대시보드)

- 차트: `recharts` 유지
- 지도: `maplibre-gl` 또는 기존 지도 훅 재사용
- 데이터 패칭: `@tanstack/react-query`
- UI 컴포넌트: 기존 Radix/shadcn 계열 유지
- 로딩/빈상태/오류상태를 모든 위젯에 일관 적용
